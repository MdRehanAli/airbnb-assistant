# OneLittleWeb Web Developer Hiring Task

This project is a frontend implementation of the provided Figma design as part of the Web Developer hiring task for OneLittleWeb.

The goal of this task was to convert the provided UI design into a fully functional and responsive webpage using Next.js while maintaining design accuracy, clean code structure, and responsive behavior.

## ⚙️ Installation & Setup

# Clone the repository:
git clone https://github.com/MdRehanAli/airbnb-assistant.git

# Navigate to the project directory:
cd airbnb-assistant

# Install dependencies:
npm install

# Run the development server:
npm run dev

# Open in browser:
http://localhost:3000
